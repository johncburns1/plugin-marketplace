# Product Definition Guide Skill

A conversational skill for creating comprehensive product definition documents through a structured interview process.

## What It Does

Guides you through creating a `PRODUCT.md` file that captures:

- Product vision and problem statement
- Target users, goals, and features
- Success metrics and scope boundaries
- Technical considerations and open questions

## When to Use

Use this skill when you want to:

- Create a product spec or PRD
- Define requirements for a new project
- Create a PRODUCT.md file
- Document product vision and strategy

## Output

Generates a comprehensive PRODUCT.md file with structured sections, feature prioritization (Must/Should/Nice to have), and clear success criteria.
