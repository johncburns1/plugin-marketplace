# Engineering Standards Skill

**Version**: 1.0.0
**Last Updated**: 2025-12-23

## Description

Core engineering principles and patterns for building maintainable, testable software. This skill provides Claude with expertise in:

- **Simplicity-First Philosophy**: Direct implementations over clever abstractions
- **Test-Driven Development**: TDD workflow, test organization, and coverage guidance
- **Hexagonal Architecture**: Domain-Driven Design patterns with ports and adapters

## When Claude Uses This Skill

Claude automatically activates this skill when:

- Designing new features
- Refactoring code
- Making architectural decisions
- Reviewing code structure
- Planning test strategies
