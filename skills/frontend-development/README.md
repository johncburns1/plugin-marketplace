# Frontend Development

Senior-level React and Vite development standards for building responsive web applications.

## What It Does

Provides architectural guidance for building React web applications with focus on:

- Simplicity and maintainability
- Mobile-first responsive design
- Type safety with TypeScript
- Component composition patterns

## When to Use

Use this skill when:

- Building web applications with React
- Designing component hierarchies
- Implementing responsive or mobile-first design
- Setting up a new React + Vite project
- Making frontend architectural decisions
