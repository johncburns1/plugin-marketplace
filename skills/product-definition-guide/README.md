# Product Definition Guide Skill

**Version**: 1.0.0
**Last Updated**: 2025-12-28

## Description

Conversational methodology for creating comprehensive product definition documents (PRODUCT.md files). This skill guides Claude through a structured interview process to extract:

- Product vision and problem statement
- Target users and goals
- Feature prioritization (Must/Should/Nice to have)
- Success metrics and scope boundaries
- Technical considerations and open questions

## When This Skill Activates

Activate this skill when the user:

- Wants to create a product spec or PRD
- Is starting a new project and defining requirements
- Asks to create a PRODUCT.md file
- Needs help organizing product thinking
- Wants to document product vision and strategy

## Output

Generates a comprehensive `PRODUCT.md` file with:

- 12 structured sections covering vision through milestones
- Industry best practices for product documentation
- Checkbox lists for features and open questions
- Clear scope boundaries and success criteria
- Template-driven consistency
