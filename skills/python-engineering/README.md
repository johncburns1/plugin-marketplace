## Description

Python-specific tooling and practices for Python 3.13+ projects using modern development standards. This skill provides Claude with expertise in:

- **uv**: Modern package management and virtual environments
- **Ruff**: Fast linting and formatting
- **mypy**: Static type checking with strict configuration
- **pytest**: Testing framework with coverage reporting
- **pre-commit**: Automated checks before commits

## When Claude Uses This Skill

Claude automatically activates this skill when:

- Setting up Python projects
- Working with Python code
- Configuring Python tooling
- Writing tests
- Adding type annotations
